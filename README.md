# Apology Website

A simple apology website with frontend and backend built using Express.

## How to run

1. Open a terminal in `C:\Users\tarig\Documents\apology-website\sorry-app`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in your browser

## What is included

- Frontend: `public/index.html`, `public/styles.css`, `public/script.js`
- Backend: `server.js`
- API routes:
  - `GET /api/details`
  - `POST /api/message`
